# 12. hét – Vizsgafelkészítés

> **Cél:** Az eddigi anyag áttekintése, gyakorlás és felkészülés a vizsgára.

## Összefoglaló – HTML

| Téma | Főbb elemek |
|------|-------------|
| Dokumentum szerkezet | `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` |
| Szöveg | `<h1>`–`<h6>`, `<p>`, `<strong>`, `<em>`, `<br>` |
| Listák | `<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>` |
| Linkek és képek | `<a href>`, `<img src alt>` |
| Táblázatok | `<table>`, `<tr>`, `<th>`, `<td>`, `<thead>`, `<tbody>` |
| Űrlapok | `<form>`, `<input>`, `<textarea>`, `<select>`, `<button>` |
| Szemantikus elemek | `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>` |

---

## Összefoglaló – CSS

| Téma | Főbb fogalmak |
|------|---------------|
| Szelektorok | elem, osztály (`.`), azonosító (`#`), kombinált |
| Box modell | `margin`, `padding`, `border`, `box-sizing` |
| Megjelenítés | `display`, `position`, `float`, `z-index` |
| Flexbox | `display: flex`, `justify-content`, `align-items` |
| Reszponzív | `@media`, viewport, mobile-first |
| Haladó | `transition`, `transform`, `box-shadow`, CSS változók |

---

## Vizsgára készülés tippei

1. **Gyakorolj kódolást** – Ne csak olvasd az anyagot, írd meg a kódot kézzel
2. **Használd a fejlesztői eszközöket** – F12 a böngészőben
3. **Nézd át a korábbi feladatokat** – Különösen amelyek nehezebben mentek
4. **Figyelj a részletekre** – Záró tagek, pontosvesszők, idézőjelek
5. **Teszteld minden böngészőben** – Chrome, Firefox minimum

---

## Vizsga típusfeladatok

A vizsgán egy komplett weboldalt kell készítened, ami tartalmaz:

- Szemantikus HTML szerkezet
- Navigáció és láblécek
- Többféle szöveges tartalom (címsorok, bekezdések, listák)
- Képek és linkek
- Táblázat vagy űrlap
- CSS stílusok (színek, betűtípusok, elrendezés)
- Reszponzív megjelenés
