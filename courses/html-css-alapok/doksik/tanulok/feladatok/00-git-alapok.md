# Feladatok – 0. hét: Git és GitHub alapok

> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz
> A megoldásokat commitold és pushold a GitHub repódba.

---

### 0.1 – Első repo ⭐
Hozz létre egy `index.html` fájlt „Hello World" tartalommal. Commitold és pushold GitHubra.

### 0.2 – Több commit ⭐
Módosítsd az `index.html` fájlt háromszor (pl. adj hozzá címsort, bekezdést, listát), és mindegyik módosítás után készíts egy külön commitot értelmes üzenettel. Pushold az eredményt.

### 0.3 – Git napló ⭐⭐
Futtasd a `git log --oneline` parancsot a terminálon, és másold be a kimenetét egy `git_naplo.txt` fájlba. Commitold és pushold.
