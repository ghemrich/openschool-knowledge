# Feladatok – 12. hét: Vizsgafelkészítés

> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz
> A megoldásokat commitold és pushold a GitHub repódba.

---

### 12.1 – HTML ismétlés ⭐
Készíts egy oldalt, amelyen használod az összes tanult HTML elemet: címsorok, bekezdések, listák, linkek, képek, táblázat, űrlap, szemantikus elemek. Ez egy összefoglaló „cheat sheet" oldal legyen.

### 12.2 – CSS ismétlés ⭐
Készíts egy CSS-t az előző oldalhoz, amelyben használod az összes tanult CSS technikát: szelektorok, box modell, Flexbox, média query, transition, változók, árnyékok.

### 12.3 – Időre való kódolás ⭐⭐
Készíts „nulláról" egy egyszerű oldalt 30 perc alatt: fejléc, navigáció, kétoszlopos tartalom, lábléc. Mérd az időt és gyakorolj, amíg beleférsz.

### 12.4 – Hibakeresés ⭐⭐
Az alábbi hibás kódot javítsd ki (készítsd el a fájlt ezzel a tartalommal, majd javítsd):
```html
<html>
<head><title>Teszt</head>
<body>
<h1>Cím<h1>
<p class=szoveg>Bekezdés<p>
<img src=kep.jpg>
<a href="oldal.html>Link</a>
</body>
```

### 12.5 – Vizsga szimuláció ⭐⭐⭐
Készíts egy komplett weboldalt 60 perc alatt az alábbi követelmények alapján:
- Reszponzív, szemantikus HTML5
- Fejléc logóval és navigációval
- Legalább 2 fő szekció tartalommal
- Táblázat vagy űrlap
- Kártya elemek Flexbox-szal
- Konszisztens stílusok CSS változókkal
- Mobilbarát megjelenés
