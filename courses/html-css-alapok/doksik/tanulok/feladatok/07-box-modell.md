# Feladatok – 7. hét: CSS box modell

> A megoldásokat commitold és pushold a GitHub repódba.

---

### 7.1 – Box modell demo ⭐
Készíts egy oldalt 3 div elemmel, amelyeknek eltérő `margin`, `padding` és `border` értékei vannak. F12-vel nyisd meg a fejlesztői eszközöket és nézd meg a box modell vizualizációt.

### 7.2 – Kártya komponens ⭐
Készíts egy kártya stílusú elemet, amelynek van:
- Kerete (`border`)
- Belső térköze (`padding`)
- Lekerekített sarkai (`border-radius`)
- Külső térköze a többi elemtől (`margin`)

### 7.3 – Border-box ⭐⭐
Készíts két egyforma szélességű dobozt: az egyiknél használd a `box-sizing: content-box` (alapértelmezett), a másiknál a `box-sizing: border-box` beállítást. Adj mindkettőnek azonos padding-et és border-t, demonstráld a különbséget.

### 7.4 – Kártya galéria ⭐⭐
Készíts 6 kártya elemet képekkel, címmel és szöveggel. A kártyák legyenek egyforma méretűek, rendezett elrendezésben, és egymástól egyenlő távolságra. Használd a `box-sizing: border-box` beállítást.

### 7.5 – Magazin elrendezés ⭐⭐⭐
Készíts egy magazin stílusú oldalt, ahol a cikkek keretes dobozokban vannak, a fő cikk nagyobb helyet kap, és a sidebar más háttérszínnel van kiemelve. Figyelj a konzisztens térközökre.
