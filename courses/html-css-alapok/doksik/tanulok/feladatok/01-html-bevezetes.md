# Feladatok – 1. hét: HTML bevezetés

> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz
> A megoldásokat commitold és pushold a GitHub repódba.

---

### 1.1 – Alapszerkezet ⭐
Hozz létre egy `index.html` fájlt a teljes HTML alapszerkezettel (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`). Az oldalnak legyen címe (`<title>`) és egy `<h1>` címsor.

### 1.2 – Bemutatkozó oldal ⭐
Készíts egy oldalt, amely tartalmaz:
- Egy `<h1>` címsort a neveddel
- Egy `<h2>` alcímet: „Rólam"
- Legalább két `<p>` bekezdést magadról

### 1.3 – Címsor hierarchia ⭐⭐
Készíts egy oldalt, ahol bemutatod a hat címsorszintet (`<h1>`–`<h6>`). Minden címsor alatt írj egy rövid bekezdést, ami elmagyarázza, mire való az adott szint.

### 1.4 – Szövegformázás ⭐⭐
Készíts egy oldalt, amelyben használod a `<strong>`, `<em>`, `<mark>`, `<del>`, `<sub>` és `<sup>` elemeket. Mindegyikre írj egy példamondatot.

### 1.5 – Több oldalas webhely ⭐⭐⭐
Készíts két HTML fájlt (`index.html` és `about.html`), amelyek egymásra hivatkoznak. Mindkét oldalon legyen fejléc, tartalom és egy link a másik oldalra.
