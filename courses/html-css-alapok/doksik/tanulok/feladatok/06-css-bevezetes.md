# Feladatok – 6. hét: CSS bevezetés

> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz
> A megoldásokat commitold és pushold a GitHub repódba.

---

### 6.1 – Első stíluslap ⭐
Készíts egy HTML oldalt és egy külső `style.css` fájlt. Állítsd be a háttérszínt, a szöveg színét és a betűtípust a `body` elemre.

### 6.2 – Szelektorok ⭐
Készíts egy oldalt, amelyen bemutatod a különböző szelektorokat. Stílusozz:
- Elem szelektorral (`p`, `h1`)
- Osztály szelektorral (`.kiemelt`)
- Azonosító szelektorral (`#fo-cim`)

### 6.3 – Színek és háttér ⭐⭐
Készíts egy színpaletta oldalt, ahol legalább 6 div jelenik meg más-más háttérszínnel. Használj különböző színformátumokat: névvel (`red`), hexadecimálisan (`#ff0000`), RGB-vel (`rgb(255,0,0)`) és HSL-lel.

### 6.4 – Szöveg stílusok ⭐⭐
Készíts egy oldalt, amelyen bemutatod a szövegformázási lehetőségeket:
- `font-size`, `font-family`, `font-weight`
- `text-align`, `text-decoration`, `text-transform`
- `line-height`, `letter-spacing`

### 6.5 – Stílusos blog ⭐⭐⭐
Készíts egy blog oldalt (HTML + CSS), ahol a fejléc, navigáció, cikkek és lábléc eltérő stílusúak. Használj osztályokat és különféle szelektorokat. Az egész oldal legyen vizuálisan vonzó.
