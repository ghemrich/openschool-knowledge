### 11.1 – CSS változók ⭐
Készíts egy oldalt, ahol CSS változókat (custom properties) definiálsz a `:root`-ban a színekhez és térközökhöz. Használd ezeket a változókat az egész oldalon. Változtasd meg egy szín változót és figyeld meg, hogyan frissül az egész oldal.

### 11.2 – Hover effektek ⭐
Készíts 4 gombot, amelyeknek különböző hover effektjei vannak `transition`-nel:
- Háttérszín váltás
- Méret növekedés (`transform: scale`)
- Árnyék megjelenés (`box-shadow`)
- Szín invertálás

### 11.3 – Kártya hover animáció ⭐⭐
Készíts kártya elemeket, amelyek hover-re:
- Enyhén felemelkednek (`translateY`)
- Árnyékuk megnő
- A kép enyhén nagyítódik
- Az átmenet sima legyen (`transition`)

### 11.4 – Sötét mód ⭐⭐
Készíts egy oldalt, amely CSS változókkal definiálja a színsémát. Hozz létre egy alternatív osztályt (`.sotet`), amely a sötét mód színeit tartalmazza. JavaScript nélkül is működhet, ha a `body`-ra rakod az osztályt.

### 11.5 – Komplett weboldal projekt ⭐⭐⭐
Készíts egy komplett, reszponzív weboldalt egy választott témában (étterem, portfólió, blog). Kötelező elemek:
- Szemantikus HTML
- CSS változók a színekhez
- Flexbox elrendezés
- Reszponzív design media query-kkel
- Átmenetek és árnyékok
- Legalább 3 oldal (index, about, contact)
