# 1. hét – HTML bevezetés és dokumentum szerkezet

> **Határidő:** a következő óra előtt
> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

## 1.1 – Bemutatkozó oldal ⭐
Készíts egy weboldalt (`feladat1.html`), amely tartalmazza a neved egy `<h1>` címsorban, és legalább 2 bekezdést (`<p>`) magadról.

## 1.2 – Címsorok és bekezdések ⭐
Készíts egy weboldalt (`feladat2.html`) a kedvenc tantárgyaidról. Használj `<h1>` főcímsort, legalább 3 alcímsort (`<h2>`), és minden alcím alatt egy bekezdést.

## 1.3 – Formázott szöveg ⭐⭐
Készíts egy weboldalt (`feladat3.html`) egy rövid hírről vagy történetről. Használj:
- `<strong>` a fontos szavakhoz
- `<em>` a kiemelt részekhez
- `<mark>` a kulcsszavakhoz
- `<hr>` elválasztóvonalat a szekciók között

## 1.4 – Személyes portfólió (alap) ⭐⭐
Készíts egy weboldalt (`feladat4.html`) az alábbi szerkezettel:
- `<h1>` – a neved
- `<h2>` – „Rólam", „Hobbijaim", „Elérhetőség"
- Minden szekció alatt bekezdések a megfelelő tartalommal
- Legalább egy `<br>` sortörés és egy `<hr>` elválasztó

## 1.5 – Receptoldal ⭐⭐⭐
Készíts egy weboldalt (`feladat5.html`) a kedvenc recepteddel:
- `<h1>` – az étel neve
- `<h2>` – „Hozzávalók" szekció
- `<h2>` – „Elkészítés" szekció
- A hozzávalókat és az elkészítés lépéseit bekezdésekben írd le
- Használj `<strong>`, `<em>` kiemeléseket
- A dokumentum `<title>` eleme tartalmazza az étel nevét
