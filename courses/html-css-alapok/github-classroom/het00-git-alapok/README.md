# 0. hét – Git és GitHub alapok

> **Határidő:** a következő óra előtt
> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

## 0.1 – Első weboldal ⭐
Hozz létre egy `index.html` fájlt, és írd bele a következőket:
```html
<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Első oldalam</title>
</head>
<body>
    <h1>Helló, világ!</h1>
    <p>Ez az első weboldalam.</p>
</body>
</html>
```
Commitold és pushold!

## 0.2 – Több commit ⭐
Módosítsd az `index.html` fájlt legalább 3 alkalommal, minden módosításnál készíts egy commitot értelmes üzenettel. Például:
1. „Cím hozzáadása"
2. „Bekezdés módosítása"
3. „Új tartalom hozzáadása"

## 0.3 – Git napló ⭐⭐
Futtasd a `git log --oneline` parancsot, és mentsd az eredményt egy `git_naplo.txt` fájlba. Commitold és pushold!
