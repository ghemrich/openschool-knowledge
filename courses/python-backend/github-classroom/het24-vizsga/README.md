> A vizsga feladatok a vizsga napján kerülnek kiosztásra.

## Vizsga információ

- **Időtartam:** 240 perc
- **Összpontszám:** 60 pont
- **Segédeszköz:** saját gép, VS Code, internet (dokumentáció)
- **Nem használható:** AI eszközök, más személy segítsége

## Felépítés

| Feladat | Pont | Tartalom |
|---------|------|----------|
| 1. feladat | 10 | Pydantic sémák + segédfüggvények |
| 2. feladat | 20 | SQLAlchemy adatbázis + CRUD API |
| 3. feladat | 10 | JWT authentikáció |
| 4. feladat | 10 | Adatfeldolgozás (feldolgozas.py) |
| 5. feladat | 10 | Docker + tesztelés |

## Felkészülés

1. Oldd meg a próbavizsgákat 240 perc alatt
2. Gyakorold a teljes projekt felépítést a nulláról (SQLAlchemy + Auth + Docker)
3. Készíts sablon projektet amit gyorsan tudsz módosítani
